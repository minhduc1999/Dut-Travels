# Dut Travels 🌍

Dut Travels is a personal travel website built with Walgo and deployed on Walrus decentralized storage.

The goal of this project is to explore how travel memories, photos, and personal stories can be preserved permanently onchain.

🔗 Live website  
https://duttravels.wal.app

---

## ✨ About the Project

Travel memories are often stored on centralized platforms that may disappear over time.

Dut Travels is an experiment to store and present travel experiences using decentralized infrastructure.

The website includes:
- Travel destinations
- Photos from different places
- Personal travel stories

All deployed using Walgo and stored on Walrus Protocol.

---

## 🛠 Built With

- HTML
- CSS
- JavaScript
- Walgo
- Walrus Protocol

---

## 🚀 Deployment

This website was published using Walgo and stored permanently on Walrus decentralized storage.

No traditional hosting or servers are required.

---

## 🌏 Concept

The idea behind this project is simple:

> Travel memories should last forever.

By using decentralized storage, this website demonstrates how personal stories and digital memories can live permanently onchain.
